# GOG Extract

Script for unpacking GOG Linux installers.

Explanation of how it works is in my [blog post].

## Usage

`gogextract.py <input file> <output dir>`

Output files will be named `unpacker.sh`, `mojosetup.tar.gz` and `data.zip`.
